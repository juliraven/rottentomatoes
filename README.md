# Analiza sentymentu recenzji filmowych 

## Opis projektu
Niniejszy projekt powstał w ramach pracy inżynierskiej "Analiza sentymentu recenji filmowych". Jest to aplikacja napisana w języku Python,
która pozwala na przetestowanie zbudowanych w ramach pracy modeli. Użytkownik ma opcję wprowadzenia własnej recenzji w języku angielskim i przewidzenia jej
sentymentu z wykorzystaniem jednego z dwóch algorytmów.

## Link do aplikacji  
[https://sentiment-analysis-rotten-tomatoes.streamlit.app/](https://sentiment-analysis-rotten-tomatoes.streamlit.app/)

## Użyte biblioteki
- streamlit
- pandas
- plotly
- seaborn
- nltk
- tensorflow
